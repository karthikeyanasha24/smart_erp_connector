Connect Google Sheets to your SmarterP ERP backend for live KPIs, branch data, transactions, and AI queries.

Quick start

  1. Open a Google Sheet with the add-on installed.
  2. 🔗 SmarterP → Open SmarterP Sidebar (or Settings from the More tab).
  3. Enter Backend URL (HTTPS, no trailing slash), e.g. https://smart-erp-backend-aa9k.onrender.com
  4. Enter your SmarterP email and password → Test Connection → Save & Connect.
  5. Use sidebar tabs: AI, Pull, Explorer.

Requirements

  • SmarterP backend on HTTPS (public URL; not localhost)
  • Valid backend user account
  • Approve Google permissions when prompted

Periods

Today · MTD (month to date) · YTD · QTD · 7d · 30d — select pill tabs at the top of the sidebar.

Troubleshooting

  • URL not configured — Complete Settings first.
  • Login failed (401) — Check email/password on the backend.
  • API 404 — Ensure add-on is updated; paths use /analytics/....
  • Slow or timeout — Wake cloud backend (open /health in browser).
  • Dialog permission error — Re-authorize the add-on after updates.
